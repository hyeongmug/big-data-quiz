# 📊 빅데이터 분석기사 기출문제 풀이 웹 앱

빅데이터 분석기사 필기 시험 대비를 위한 웹 기반 기출문제 풀이 애플리케이션입니다.  
HTML, CSS, JavaScript만으로 구현되어 별도의 서버 없이 브라우저에서 바로 실행 가능하며, GitHub Pages 등을 통해 쉽게 배포할 수 있습니다.

## ✨ 주요 기능

### 1. 📝 기출문제 풀이
*   **회차별 문제 제공**: 여러 회차의 기출문제를 선택하여 풀 수 있습니다.
*   **자동 저장**: 문제를 풀면 진행 상황(마지막 문제 위치, 선택한 답안)이 브라우저에 자동 저장되어, 나중에 다시 접속해도 이어서 풀 수 있습니다.
*   **정답 및 해설 확인**: 문제를 풀면 즉시 정답 여부와 상세한 해설을 확인할 수 있습니다.

### 2. 🎯 학습 모드 다양화
*   **과목별 풀이**: 1과목부터 4과목까지 특정 과목의 문제만 모아서 집중적으로 학습할 수 있습니다.
*   **🎲 랜덤 모의고사**: 전체 문제 중 20문제를 무작위로 추출하여 실제 시험처럼(타이머 제공, 정답 숨김) 풀어볼 수 있습니다.
*   **⭐ 오답 노트 (즐겨찾기)**: 헷갈리거나 틀린 문제를 별표(★)하여 나중에 모아볼 수 있습니다.

### 3. ⚙️ 편의 기능 및 설정
*   **다크 모드**: 눈의 피로를 줄여주는 다크 모드를 지원합니다.
*   **글자 크기 조절**: 문제와 보기의 글자 크기를 원하는 대로 조절할 수 있습니다.
*   **키보드 단축키**: 마우스 없이 키보드(화살표, 숫자키)만으로 문제를 풀 수 있습니다.
*   **모바일 스와이프**: 모바일 환경에서 화면을 좌우로 밀어 이전/다음 문제로 이동할 수 있습니다.

### 4. 📊 학습 통계
*   전체 진행률, 정답률, 과목별 성취도를 그래프로 시각화하여 보여줍니다.

## 🚀 실행 방법

### 로컬에서 실행하기
1.  이 저장소를 클론(Clone)하거나 다운로드합니다.
2.  `index.html` 파일을 웹 브라우저(Chrome, Edge 등)로 엽니다.

### GitHub Pages 배포하기
1.  GitHub 저장소에 코드를 업로드합니다.
2.  저장소 **Settings** > **Pages** 메뉴로 이동합니다.
3.  **Source**를 `Deploy from a branch`로 설정하고, **Branch**를 `main` (또는 `master`)으로 선택 후 저장합니다.
4.  잠시 후 생성된 URL을 통해 어디서든 접속할 수 있습니다.

## 📂 폴더 구조

```text
/
├── index.html          # 메인 페이지 (홈 화면 및 퀴즈 UI)
├── style.css           # 스타일 시트 (다크 모드, 반응형 디자인 포함)
├── script.js           # 핵심 로직 (문제 로드, 상태 저장, 모드 관리 등)
├── data/               # 데이터 폴더
│   ├── index.json      # 기출문제 회차 목록 메타데이터
│   ├── exam_10.json    # 제10회 기출문제 데이터
│   └── ...             # 추가 회차 데이터
└── README.md           # 프로젝트 설명 문서
```

## 🛠️ 기술 스택
*   **Frontend**: HTML5, CSS3, JavaScript (Vanilla JS)
*   **Data**: JSON
*   **Storage**: LocalStorage (브라우저 내 데이터 저장)

## 📝 데이터 추가 방법
새로운 기출문제를 추가하려면:
1.  `data/` 폴더에 `exam_XX.json` 파일을 생성하고 문제 데이터를 입력합니다. (기존 파일 형식 참고)
2.  `data/index.json` 파일에 해당 회차 정보를 추가합니다.
